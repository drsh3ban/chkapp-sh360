# AutoCheck Pro - Web Application

تطبيق ويب احترافي لإدارة نظام تشييك السيارات

## التشغيل المحلي

```bash
# تثبيت التبعيات
npm install

# تشغيل خادم التطوير
npm run dev

# بناء للإنتاج
npm run build
```

## البنية

- `src/` - الكود المصدري
  - `components/` - المكونات القابلة لإعادة الاستخدام
  - `pages/` - صفحات التطبيق
  - `services/` - خدمات API
  - `store/` - إدارة الحالة
  - `utils/` - دوال مساعدة
  - `styles/` - ملفات CSS

## التقنيات

- Vite - أداة البناء
- Tailwind CSS - إطار عمل CSS
- Vanilla JavaScript - بدون إطار عمل
